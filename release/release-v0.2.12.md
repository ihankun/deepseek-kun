# Kun v0.2.12

这一版带来一个常驻的内置终端面板，并把模型端点格式细化到“按模型”级别，同时为 Write 工作台加入多 Agent 人设预设。它让“在应用里直接跑命令、按模型精确路由、按场景切换写作助手”成为现实。

### 内置终端面板

- 新增内置终端面板，可以直接在应用里开 shell、跑命令、看输出，不必切到外部终端。
- 终端基于 PTY 与主进程打通，交互体验接近原生终端。

### 按模型的端点格式

- 端点格式从“按供应商”细化到“按模型”：同一个供应商下，不同模型可以走不同的 wire format（如 chat_completions 与 messages），缺省继承供应商设置。
- 这让混合接入多家、多形态模型的供应商配置更精确，切换模型无需重启。

### Write 多 Agent 人设

- Write 工作台新增 Agent 人设预设，可以按写作场景切换不同风格的助手。

### 升级说明

- 从 `v0.2.11` 升级可直接通过 GUI 更新。
- 升级后可以打开内置终端跑一条命令试试；如果你接入了多形态模型的供应商，留意新的“按模型端点格式”选项。

### 总结

v0.2.12 把“终端、端点、人设”三件实用能力补齐：内置终端让操作闭环留在应用内，按模型端点让多模型路由更准，多 Agent 人设让写作更贴场景。
