# Kun v0.2.13

这是一次聚焦 SDD（需求驱动开发）的小版本：把 PM 技能框架接进“新建需求”的流程，让需求的发现、结构化、计划与验收更有方法可循，同时顺手补了快捷键和设置归一化的细节。

### PM 技能框架接入新建需求流

- 引入 PM 技能框架（发现 / 结构化 / 风险 / 计划 / 验收等阶段），并接进新建需求的工作流。
- 在 discover / structure / risk 阶段以助手按钮形式提供引导，在 plan / verify 阶段把对应方法注入到提示词里。
- 通过发送时的门控避免过期引导泄漏，让框架只在合适的阶段起作用。

### 快捷键与设置

- 快捷键支持更多常见的“放大”按键变体，不同键盘布局都能触发。
- 调整了设置归一化逻辑，配置写入和校验更稳。

### 升级说明

- 从 `v0.2.12` 升级可直接通过 GUI 更新，无需额外操作。

### 总结

v0.2.13 让 SDD 流程“有章法”：把 PM 方法论接进新建需求的各个阶段，配合快捷键和设置细节的修正，需求侧体验更成熟。
