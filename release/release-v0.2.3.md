# DeepSeek GUI v0.2.3

这一版在 v0.2.2 的基础设施补强之后，把重心拉回到能力打通上：让 Skill 真正从 GUI 贯穿到 Kun 运行时，同时顺手打磨了定时任务和扩展市场的界面。它是一次“把骨架接上电”的版本。

### Skill 能力从 GUI 贯穿到运行时

- 新增 Skill 服务层，并把它从 GUI 一路接到 Kun 运行时，技能不再只是配置项，而是能被实际调度。
- 补齐了 Kun 侧的 Skill 路由与运行时更新，让模型在合适的时机拉取并使用对应技能。
- 配套调整了 Kun 配置与工具处理逻辑，让技能、工具和运行时的衔接更顺。

### 定时任务与扩展市场界面打磨

- 更新了 schedule 运行时的相关行为，定时任务的执行链路更清晰。
- 扩展市场（marketplace）UI 做了一轮调整，浏览和管理扩展更直观。

### 升级说明

- 从 `v0.2.2` 升级可直接通过 GUI 更新，无需额外操作。
- 如果你之前手动维护过 Kun 配置，建议留意本版对配置结构和工具处理的调整，尽量交给 GUI 接管。

### 总结

v0.2.3 是一次“能力接线”的版本：Skill 第一次完整地从界面走到运行时，定时任务和扩展市场的体验也更顺手，为后续更多 Agent 能力铺好了路。
