# DeepSeek GUI v0.2.4

这一版继续打磨日常交互细节，重点在三处：定时任务与时间线的行为更合理、聊天输入框能直接引用工作区文件、以及 bash 工具支持会话轮询。它让“边写边引用、边跑边观察”的体验更顺。

### 输入框直接引用工作区文件

- composer 新增工作区文件引用能力，可以把项目里的文件直接带进对话上下文。
- 配合后续的 `@` 提及与拖拽路径插入，引用文件变成自然的输入动作，而不是手动复制内容。

### bash 工具支持会话轮询

- bash 工具新增会话轮询（session polling）与元数据，长时间运行的命令可以持续观察输出，而不是一次性等待结果。
- 这让需要长跑、分阶段产出的命令在 Agent 流程里更可控。

### 定时任务与时间线行为修正

- 调整了定时任务与时间线的行为，触发和展示更符合预期。
- 改进了运行时清理逻辑，并补充了相关演示，减少残留进程和状态。

### 升级说明

- 从 `v0.2.3` 升级可直接通过 GUI 更新。
- 升级后建议试一下在输入框里引用工作区文件，以及用 bash 工具跑一个长命令，确认轮询行为符合预期。

### 总结

v0.2.4 把“引用文件”和“观察长任务”两件高频小事做顺了，配合定时任务行为修正，整体日常体验更连贯。
