# DeepSeek GUI v0.2.5

这是一次偏交互打磨的小版本，重点在聊天输入区、项目侧栏以及应用外壳与文件处理上做了一轮细节优化，让整体界面更稳、更跟手。

### 聊天输入与项目侧栏优化

- 进一步打磨了 chat composer 的交互细节，输入、切换和发送更顺手。
- 调整了项目侧栏（projects section）的展示与分组逻辑，定位项目和会话更清晰。

### 应用外壳与文件处理

- 更新了 app shell，整体窗口与布局表现更一致。
- 改进了文件处理逻辑，读写和引用文件时更稳。

### 升级说明

- 从 `v0.2.4` 升级可直接通过 GUI 更新，无需额外操作。

### 总结

v0.2.5 不主打新功能，是一次把输入区、侧栏和外壳打磨得更顺的过渡版本，适合平滑跟进。
