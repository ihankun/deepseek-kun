# DeepSeek GUI v0.2.6

这一版集中修界面观感和浮层交互：用自定义窗口控件替换了系统标题栏覆盖层，解决深色背景下标题栏不匹配的问题，同时让 FloatingComposer 的浮层子菜单定位更准。

### 窗口控件与深色标题栏修复

- 用自定义窗口控件替换了 `titleBarOverlay`，修复深色主题下标题栏背景与应用界面不一致的问题。
- 窗口控件的观感在浅色与深色主题下都更统一。

### FloatingComposer 浮层交互

- 实现了浮动输入框（FloatingComposer）子菜单的定位逻辑，弹出菜单不再错位或被裁剪。
- 选择 reasoning 选项后会自动关闭对应菜单，交互更干脆。

### 升级说明

- 从 `v0.2.5` 升级可直接通过 GUI 更新。
- 如果你之前在深色主题下觉得标题栏“顶部一条颜色不对”，这一版就是针对这个问题的修复。

### 总结

v0.2.6 是一次观感与浮层交互的修复版本，深色主题下的窗口表现和浮动输入框的菜单定位都更精细。
