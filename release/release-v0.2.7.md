# DeepSeek GUI v0.2.7

这是一次内容很密的稳定性与体验大版本。它在防白屏、原子写、系统托盘、自定义快捷键和输入体验等多个方向同时发力，把日常使用中那些“偶发但很烦”的问题集中收掉，是 v0.2 系列里很关键的一块“地基加固”。

### 防白屏与运行时诊断

- 新增顶层 Error Boundary，未捕获的渲染错误不再让整个界面变白屏，而是给出可恢复的提示。
- 稳定了消息时间线的渲染，长会话滚动和更新更平稳。
- 改进了运行时错误诊断：探测失败时会在离线 hero 区直接显示具体的运行时错误，而不是含糊的“连不上”。

### 数据写入更安全

- `FileMemoryStore` 与设置存储改为原子写，避免写一半被中断导致文件损坏。
- 专门处理了 Windows 下文件被占用时的原子写场景，减少“保存失败/配置丢失”的情况。
- 启动时会迁移旧版设置目录，老用户升级后配置自动归位。

### 系统托盘与品牌化

- 使用专用的托盘图标，并设置 `productName`，让 Windows 系统托盘显示项目名称。
- 托盘与应用图标改为通过 `createFromBuffer` 加载，确保打包进 `app.asar` 后依然能正常显示。

### 自定义快捷键与输入体验

- 新增可自定义的键盘快捷键体系。
- 支持日期等模板快捷展开，并接入 Markdown 编辑器。
- composer 支持把拖入的文件路径直接插入对话；占位文字与输入内容在高度上垂直居中；输入框内边距更舒服。
- 引入工作区根目录的去重标识，侧栏分组合并更准确。

### 设置、供应商与工具

- 新增全局 code 提示词前缀设置（closes #89）。
- 自定义 OpenAI 兼容供应商在调用时会保留供应商上下文（#26）。
- 设置补丁 schema 支持部分供应商资料与快捷键绑定的局部更新。
- bash 工具的 poll 会按 `yield_seconds` 阻塞等待；web_fetch 对未知长度响应改用流式并保留 content-length 守卫，过大响应会被截断而不是直接失败。

### 其他修复

- 修复点击左侧栏定时任务时聊天记录消失的问题。
- 防止 GUI 管理的 instructions 泄漏进聊天界面显示。
- 从磁盘刷新未完成的 SDD 草稿；`<html lang>` 跟随当前 i18n 语言（a11y）。
- 发布构建启用 macOS 签名并要求安全时间戳。

### 升级说明

- 从 `v0.2.6` 升级可直接通过 GUI 更新，首次启动会自动迁移旧版设置目录。
- 升级后可以去设置里看看新的“自定义快捷键”，并确认托盘图标显示正常。

### 总结

v0.2.7 是 v0.2 系列里分量很重的一版：它没有改变产品方向，但把白屏、配置损坏、托盘显示、输入交互等一大批稳定性与体验问题一次性补齐，让 DeepSeek GUI 作为日常工具更经得起用。
